Uruchomienie programu przez Jupyter Notebook

```
$ git clone https://github.com/jedrzejd/Mask-Detector.git
$ cd Mask-Detector
$ python3 -m venv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ jupyter notebook Covid_Analysis.ipynb
```
